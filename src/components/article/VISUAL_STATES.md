## Interactive Behaviors

### EditableContentSection

1. **Click Edit button**
   - Component transitions to edit mode
   - Input/Textarea receives focus
   - Content is selected for easy replacement
   - Save and Cancel buttons appear

2. **Type in input**
   - Character count updates in real-time
   - Color changes when nearing limit (80% = amber, 100% = red)
   - Dirty state badge appears if content differs from original

3. **Press Escape**
   - Reverts to original content
   - Exits edit mode
   - Removes dirty state

4. **Press Cmd+Enter**
   - Triggers save if content has changed
   - Shows loading state
   - On success, shows success badge for 2 seconds

5. **Click Save**
   - Same behavior as Cmd+Enter
   - Disabled if no changes or over character limit

### EditableKeyInsights

1. **Click Edit button**
   - All insights become editable
   - Add Insight button appears
   - First insight receives focus

2. **Add new insight**
   - New empty textarea appears
   - Automatically focused
   - Numbered sequentially
   - Add button disabled when max reached

3. **Remove insight**
   - Insight smoothly animates out
   - Remaining insights renumbered
   - Delete button disabled when only 1 insight remains
